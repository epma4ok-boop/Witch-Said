# 🧙‍♀️ Ведьма сказала — магический шар предсказаний для Telegram

[![Telegram](https://img.shields.io/badge/Telegram-Web%20App-blue)](https://t.me/your_bot)
[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)

Мини-приложение для Telegram с атмосферной магией, саркастичными предсказаниями и красивой анимацией. Нажми на шар — и узнай своё будущее.

## ✨ Особенности

- 🎨 Магический UI с анимациями и свечением
- 🌍 Поддержка русского и английского языков
- 📱 Полная интеграция с Telegram WebApp (хаптика, темы)
- 🎭 4 категории предсказаний: всё, любовь, работа, деньги
- 📤 Шеринг предсказаний в Telegram
- ⚡ Работает в браузере (fallback-режим)
