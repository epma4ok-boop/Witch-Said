# Ведьма сказала

Дерзкий мемный Telegram Mini App с ироничными предсказаниями.

## Структура

- `index.html` — основной интерфейс mini app
- `app/data/predictions.json` — 100 фраз ведьмы
- `api/index.py` — простая serverless-функция для выдачи случайной фразы
- `vercel.json` — роутинг для деплоя на Vercel
- `requirements.txt` — заглушка под Python runtime

## Как запустить

### Локально

Открой `index.html` в браузере или подними статический сервер.

### Деплой на Vercel

1. Создай новый репозиторий или залей файлы в существующий.
2. Подключи репозиторий к Vercel.
3. Убедись, что корень проекта — папка с этими файлами.
4. После деплоя укажи URL в BotFather для Telegram Mini App.
