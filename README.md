# Вселенная говорит — Telegram Mini App

Мистическое приложение-предсказатель с анимированным солнечным затмением.

## Запуск локально

```bash
npm install
npm run dev
```

## Деплой на Vercel

1. Загрузи файлы на GitHub
2. Подключи репозиторий в Vercel
3. Vercel автоматически обнаружит Vite и задеплоит
